## Setup & Run

Follow these steps to get the project up and running on your local machine.

### Prerequisites

- **Node.js** v14+  
- **npm** v6+ (bundled with Node.js)

### Installation

1. **Clone the repository**  
   ```bash

   cd udemy-moviedux

2. **Install dependencies**
    ```bash
    npm install

3. **Running the Development Server**
    ```bash
